# Xreviews

혹시 리뷰 볼 때, 1점부터 보시나요?

좋은 후기는 이미 충분합니다. Xreviews는 사람들이 먼저 찾는 나쁜 후기만 모읍니다.

## Stack

- Next.js App Router
- Cloudflare Workers/Pages via OpenNext Cloudflare
- Neon Postgres
- Drizzle ORM
- BetterAuth
- Resend
- Cloudflare R2
- Sentry
- GA4 + Microsoft Clarity

## Commands

```bash
pnpm install
pnpm dev
pnpm typecheck
pnpm build
```

## Product rules

- Positive reviews are not accepted.
- Star-rating UI is not allowed.
- Complaint tags are required.
- Reviews are pending before admin approval.
- Businesses can reply and post improvements.
- Businesses cannot pay to delete reviews.
